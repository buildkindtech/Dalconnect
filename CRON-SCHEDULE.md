## 📌 중요 참고사항

### 데이터 신선도 vs 비용
- **비즈니스**: 주 1회면 충분 (평점은 천천히 변함)
- **뉴스**: 매일 필수 (신선도가 생명)
- **차트**: 주 1회 (원본 차트도 주간 업데이트)

### 사용자 경험
- 새 뉴스 → 매일 확인 기대
- 비즈니스 평점 → 주간 차이 미미
- 쿠폰/딜 → 주 2회면 충분

### 확장성
- 도시 추가 시 (HouConnect 등): 크론잡 복제
- 비즈니스 수 증가 시: API 할당량 모니터링
- 트래픽 증가 시: 캐싱 전략 추가
