# 🎬 DalKonnect Reels 시스템 가이드

> **규칙**: 릴스 만들자고 하면 **먼저 이 파일 읽고** 카테고리 파악 → 해당 템플릿으로 바로 실행.
> 매번 새로 만들지 말 것. 새 카테고리 추가 시 여기 업데이트.

## 🗂️ 카테고리별 릴스 타입

| 카테고리 | 타입 | 스크립트 | 포맷 |
|---|---|---|---|
| 🍗 맛집 | 캐러셀 | `gen-{음식}-carousel.cjs` | 6장 슬라이드 |
| 📰 뉴스 | 단일 카드 | `gen-news-card-from-url.cjs` | 1장 이미지 |
| ⚠️ 건강/팁 | 정보 슬라이드 | `gen-health-reels.cjs` (→ 아래 참고) | 4-6장 |
| 🎵 K-POP/연예 | 영상+이미지 | `gen-kpop-reels.cjs` (수동) | MP4 |
| 🏙️ 달라스 생활 | 정보 카드 | `gen-news-card-from-url.cjs` (local) | 1장 |
| 🔥 커뮤니티 핫이슈 | 정보 슬라이드 | `gen-community-card.cjs` (→ 아래) | 4장 |

## 📐 카테고리 1: 맛집 캐러셀 🍗

**언제**: "XX 맛집 릴스 만들어줘"

### 슬라이드 구성 (6장)
```
[0] 커버: 음식 이모지 + "오늘은 XX다!" 타이틀
[1-4] 업체 카드: 실제 음식 사진 + 업체명/위치/평점
[5] CTA: "어디 가요? 댓글 남겨요 👇 dalkonnect.com"
```

### 비주얼 스타일
```
커버: 배경 dark gradient(#0d1117→#1a2332) + 주황 glow + 뱃지
업체 카드: 실제 사진 full-bleed + 하단 gradient(투명→검정97%)
accent: #FFA032 (주황)
폰트: Noto Sans KR 900 (제목), 700 (설명)
로고: 우하단 "DalKonnect.com"
```

### 캡션 템플릿
```
🍲 달라스 국밥 맛집 4선 🍲

DFW에서 진국 국물 마시고 싶을 때 여기!
1️⃣ {업체1} — {한줄설명}
2️⃣ {업체2} — {한줄설명}
3️⃣ {업체3} — {한줄설명}
4️⃣ {업체4} — {한줄설명}

어디 가요? 댓글로 알려주세요 👇
더 많은 맛집 정보 → dalkonnect.com

#달라스맛집 #DFW맛집 #달라스한인 #DFWkorean #달라스국밥 #달라스한식 #텍사스맛집 #달라스음식 #Carrollton #Plano #DalKonnect #달라스커뮤니티 #한인맛집 #DFW #달라스
```

### 실행 방법
```bash
# 1. DB에서 업체 사진 확인
node -e "require('./scripts/check-data.js')('국밥')"

# 2. 카드 생성
node scripts/gen-gukbap-carousel.cjs   # 국밥
# 새 음식: gen-gukbap-carousel.cjs 복사해서 업체/이모지/색상만 변경

# 3. 포스팅
node scripts/post-gukbap-carousel.cjs
```

## 📐 카테고리 2: 뉴스 카드 📰

**언제**: "이 뉴스 릴스/카드 만들어줘" + URL 또는 기사 내용 전달 시

### 비주얼 스타일
```
배경: 기사 OG 이미지 full-bleed
오버레이: 위→아래 gradient (투명 → 카테고리 색)
배지: 카테고리 태그 (좌상단)
헤드라인: 64px weight:900 흰색
divider: 4px accent 색
설명: 26px 2줄
출처 pill + 날짜 pill (하단)
CTA: "더 알아보기 👉 dalkonnect.com/news"
```

### 배지/색상 기준
| 카테고리 | 배지 | 배지색 | 오버레이 |
|---|---|---|---|
| 로컬 긴급 | 🚨 달라스 긴급뉴스 | `#dc2626` | `rgba(100,0,0,0.75)` |
| 건강/안전 | ⚠️ 텍사스 건강 경고 | `#d97706` | `rgba(80,40,0,0.8)` |
| 이민/비자 | 📋 이민/비자 업데이트 | `#1D4ED8` | `rgba(0,20,80,0.82)` |
| K-POP | 🎵 K-POP 소식 | `#7C3AED` | `rgba(30,0,60,0.82)` |
| 딜/할인 | 🛍️ 할인/딜 정보 | `#059669` | `rgba(0,50,20,0.82)` |
| 일반뉴스 | 📰 로컬 뉴스 | `#374151` | `rgba(0,0,0,0.8)` |

### 캡션 템플릿
```
{카테고리 이모지} {헤드라인 한 줄}

{2-3줄 핵심 요약}

전체 기사 → dalkonnect.com/news

{카테고리별 해시태그 (아래 참고)}
```

### 실행 방법
```bash
# URL 있을 때
node scripts/gen-news-card-from-url.cjs "https://..." health
# 카테고리: local | health | visa | kpop | deal | news

# 생성 파일: sns-cards/news-card-{timestamp}.png
# 포스팅
node scripts/post-news-3.cjs sns-cards/news-card-{timestamp}.png "{캡션}"
```

## 📐 카테고리 3: 건강/팁 정보 슬라이드 ⚠️

**언제**: "건강 정보 릴스", "팁 릴스", "설탕보다 나빠..." 같은 정보성 콘텐츠

### 슬라이드 구성 (5장)
```
[0] 훅 카드: 충격적인 한 줄 + 질문 형식
[1] 문제 제시: "실제로는..."
[2] 팩트 1: 습관/원인 + 의학적 근거
[3] 팩트 2: 습관/원인 + 의학적 근거
[4] CTA: "저장해두세요 🔖 + dalkonnect.com/news"
```

### 비주얼 스타일
```
배경: 다크 그라디언트 (테마 색상 기반)
건강: #0f172a → #1e3a5f (네이비)
음식팁: #0f1a0a → #1a3a10 (다크그린)
라이프스타일: #1a0a1a → #3a0f3a (퍼플)
accent: 카테고리별 (건강=#3B82F6, 음식=#10B981, 라이프=#A855F7)

훅 카드: 큰 이모지 + 질문 텍스트 + "알고 있었나요?" 뱃지
팩트 카드: 번호 원형 + 핵심 문장 + 출처 small 텍스트
CTA 카드: 저장 유도 + 팔로우 CTA + 링크
```

### 스크립트 구조 (매번 이 형식으로)
```
훅: "{충격적 사실}" 알고 있었나요?
문제: 많은 사람들이 {X}라고 생각하지만 사실은 {Y}
팩트1: [번호①] {원인/습관} → {결과} (출처: {기관})
팩트2: [번호②] {원인/습관} → {결과} (출처: {기관})
CTA: 저장해두세요 🔖 / 댓글로 본인 경험 공유 / 팔로우
```

### 캡션 템플릿
```
⚠️ {주제}: 의사가 경고하는 {N}가지

{핵심 내용 1-2줄}

① {팩트1 한 줄}
② {팩트2 한 줄}

출처: {기관명}
더 알아보기 → dalkonnect.com/news

#달라스건강 #건강정보 #당뇨예방 #건강팁 #DFW건강 #달라스한인 #건강생활 #헬스팁 #DalKonnect #한인커뮤니티
```

### 실행 방법 (현재 수동 → 스크립트 필요)
```bash
# TODO: gen-health-reels.cjs 만들어야 함
```

## 🚀 Aaron이 "릴스 만들자" 했을 때 실행 흐름

```

2. 해당 카테고리 섹션 읽기
3. 필요 데이터 수집 (URL / DB 업체 / 팩트 출처)
4. 스크립트 실행 or HTML 슬라이드 생성
5. Aaron에게 미리보기 보내기
6. 승인 후 포스팅 스크립트 실행
```

## ⚠️ 공통 규칙
- 이미지: 실제 사진 우선 (Unsplash 금지!)
- 출처 없는 건강/팩트 릴스 금지 — 반드시 NIH/Mayo/CDC 등 명기
- 업로드: `post-news-3.cjs` or `post-gukbap-carousel.cjs` 재사용
- dotenv: DB = `.env` / 포스팅 = `.env.local`
- 이미지 공개 URL: FB photo upload 거쳐야 IG 게시 가능
