# DalConnect 실제 딜 소스 리서치

**목표:** 100% 실제 딜만 수집, 자동화 가능, 정확한 URL/만료일

---

## 🎯 Option 1: 딜 Aggregator APIs (즉시 사용 가능)

### 1.1 **RapidAPI Deal Aggregators**
- **SlickDeals API** (rapidapi.com/slickdealsapi)
  - 실제 딜, 쿠폰 코드, 만료일
  - 카테고리 필터 (grocery, restaurant 등)
  - **비용:** $0-50/월 (사용량)
  - **신뢰도:** ★★★★★

- **RetailMeNot API**
  - 쿠폰 코드 + 프로모 링크
  - 검증된 딜만
  - **비용:** API 접근 제한적

### 1.2 **Affiliate Networks**
- **CJ Affiliate / ShareASale**
  - 한인 마트는 없지만 Amazon, Walmart 등
  - 실제 제품 링크, 정확한 가격
  - **수익:** Commission 가능 (5-10%)

## 🛒 Option 2: 한인 마트 직접 연동

### 2.1 **H마트 Weekly Ad**
**현황:**
- PDF/이미지로만 제공
- https://www.hmart.com/weekly-ads
- 매주 수요일 업데이트

**자동화 방법:**
1. **PDF 다운로드** (Puppeteer)
2. **OCR 텍스트 추출** (Tesseract / Google Vision API)
3. **상품명/가격 파싱** (정규식)
4. **URL:** H마트 메인 + 검색 링크
5. **만료일:** 다음 화요일

**비용:**
- Google Vision API: $1.50/1000 페이지
- 월 4번 (주 1회) = $0.01/월

**신뢰도:** ★★★★☆ (OCR 정확도 85-90%)

**구현 시간:** 4-6시간

### 2.2 **한남체인, Zion Market 등**
- 대부분 PDF/이미지 형태
- H마트와 동일한 방식

  - 해시태그별 최신 포스트 수집
  - 이미지 + 캡션 추출
  - **비용:** $49/월 (unlimited)

**장점:**
- ✅ 실제 사용자들이 공유한 딜
- ✅ 사진으로 검증 가능
- ✅ 아줌마들이 직접 공유!

**단점:**
- ❌ URL 추출 어려움 (bio 링크만)
- ❌ 만료일 불명확

**신뢰도:** ★★★★☆

### 3.2 **Facebook Groups (Graph API)**
**타겟 그룹:**
- "달라스 한인 맘카페" (Public만)
- "DFW 한인 육아"

**자동화:**
- Facebook Graph API
  - 그룹 포스트 검색 ("딜", "쿠폰", "세일")
  - **비용:** 무료 (앱 승인 필요)

**문제:**
- ❌ 대부분 Private 그룹 (접근 불가)
- ❌ API 승인 까다로움

## 💡 Option 4: 크라우드소싱 (가장 현실적)

### 4.1 **사용자 딜 제출 시스템**
**워크플로우:**
1. 사용자 "딜 공유하기" 버튼 클릭
2. 폼 작성:
   - 제목, 스토어, 할인율
   - **실제 딜 URL** (필수 - 자동 검증)
   - 만료일 (달력 선택)
   - 이미지 업로드 (선택)
3. 자동 검증:
   - URL이 실제로 작동하는지 체크
   - 이미지에서 텍스트 추출 (선택)
4. 관리자 승인 대기 (24시간 내)
5. 승인되면 자동 게시

**인센티브:**
- 포인트 시스템 (딜 제출 시 100pt)
- 상위 기여자 배지
- 월간 TOP 기여자 상품권 ($50)

**초기 시딩:**
- 직접 10-20개 실제 딜 입력
- H마트 가서 weekly ad 사진 찍기
- 친구/가족에게 딜 공유 요청

**장점:**
- ✅ 즉시 구현 가능 (2-3시간)
- ✅ 100% 실제 딜 (사용자 검증)
- ✅ 커뮤니티 참여 유도
- ✅ 확장 가능 (사용자 증가 = 딜 증가)

**단점:**
- ❌ 초기 딜 적음 (cold start)
- ❌ 관리 필요 (승인 프로세스)

**신뢰도:** ★★★★★
