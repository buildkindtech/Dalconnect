## 🚨 에러/반복 실수 — 다시는 하지 말 것

### 1. Vite 빌드 경로 실수
- **❌ 잘못**: `npx vite build --outDir dist/public` → `client/dist/public/`에 빌드됨
- **✅ 올바름**: `npx vite build` (플래그 없이) → 프로젝트 루트 `dist/public/`에 빌드됨
- **이유**: `vite.config.ts`의 `root: "client/"` 때문에 `--outDir`이 client/ 기준으로 해석됨

### 2. ESM에서 require() 사용
- **❌ 잘못**: `server/routes.ts` (ESM)에서 `require('crypto').createHash(...)` 사용
- **✅ 올바름**: 파일 상단에 `import crypto from "crypto"` 추가 후 `crypto.createHash(...)` 사용
- **증상**: 댓글 달기 → `{"success":false,"message":"require is not defined"}`

### 3. 번역 skip 조건 버그
- **❌ 잘못**: `if (/[\uAC00-\uD7AF]/.test(title)) return { title, content }` → 제목만 한글이면 본문도 번역 안 함
- **✅ 올바름**: 제목 + 본문 둘 다 한글 확인 후 스킵
  ```js
  const titleIsKorean = /[\uAC00-\uD7AF]{3,}/.test(title);
  const contentIsKorean = /[\uAC00-\uD7AF]{3,}/.test(content || '');
  if (titleIsKorean && contentIsKorean) return { title, content };
  ```
- **번역 본문 길이**: 300자 → 1500자로 확장 (너무 짧으면 요약만 됨)

### 4. DB 시간 UTC/로컬 파싱 오류
- **❌ 잘못**: `new Date(dateString)` — DB에서 오는 timestamp에 'Z' 없으면 로컬 시간으로 해석
- **✅ 올바름**: `new Date(dateString.endsWith('Z') ? dateString : dateString + 'Z')`
- **증상**: "분 전"이 "-283분 전" 같은 음수로 표시

### 5. 해시태그 하드코딩
- **❌ 잘못**: `tags: ['reddit', '달라스']` — 모든 카테고리가 동일 태그
- **✅ 올바름**: `tags: [src.category, src.sub.toLowerCase()]` — 카테고리 기반 동적 태그

### 6. 백그라운드 프로세스 결과 손실
- `command &` 로 백그라운드 실행하면 결과 확인 어려움
- 긴 배치 작업은 `exec + yieldMs`로 포그라운드 실행 or `process(poll)` 사용
